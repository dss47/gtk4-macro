\section{Module \texttt{separator}}

Le module \texttt{separator} crée une séparation visuelle horizontale ou verticale entre zones d'interface.

\subsection{Macros de Configuration}

\begin{docbox}{Macro \texttt{SEPARATOR\_CONFIG\_DEFAULT}}
\begin{lstlisting}[language=C]
#define SEPARATOR_CONFIG_DEFAULT ((separator_config){ \
    .orientation = GTK_ORIENTATION_HORIZONTAL, \
    .style = WIDGET_STYLE_CONFIG_DEFAULT, \
})
\end{lstlisting}
\end{docbox}

Cette macro initialise un séparateur prêt à l'emploi dans le cas le plus fréquent (horizontal). Elle évite d'avoir une orientation non définie au moment du rendu. La valeur de style par défaut garantit aussi une application cohérente du thème.

Même pour un composant simple, une initialisation explicite reste importante: elle supprime les ambiguïtés d'affichage et évite des anomalies d'orientation dans des layouts complexes.

\subsection{Fonctions API}

\begin{lstlisting}[language=C]
GtkWidget *create_separator(const separator_config *config) {
    GtkWidget *separator = gtk_separator_new(
        config ? config->orientation : GTK_ORIENTATION_HORIZONTAL
    );
    if (config != NULL) {
        apply_widget_style(separator, &config->style);
    }
    return separator;
}
\end{lstlisting}

La fonction conserve un comportement stable même sans configuration explicite.

Elle constitue un bon exemple de wrapper minimal: une API courte, des valeurs par défaut robustes et un point unique de personnalisation via \texttt{apply\_widget\_style}.
