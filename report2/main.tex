\section{Introduction}
Ce volume décrit l'implémentation des modules widgets du projet GTK4. Pour chaque module, on présente la configuration publique (macros et structures) puis la logique des fonctions API côté \texttt{.c}. L'objectif est de relier l'interface de programmation aux choix techniques de robustesse (valeurs par défaut, vérifications \texttt{NULL}, branchement des signaux).
