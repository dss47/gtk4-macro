\chapter{Gestion de la Fenêtre Principale (Windows)}

\section{Introduction}
La fenêtre est le composant racine de toute application GTK. Ce module permet de définir les propriétés fondamentales comme le titre, la taille par défaut et l'apparence visuelle (fond coloré ou image). Il utilise les capacités de \texttt{GtkApplicationWindow} pour une intégration parfaite dans l'environnement de bureau.

\section{Configuration de la Fenêtre (Header)}

\begin{winbox}{Structure : window\_config}
La structure \texttt{window\_config} regroupe toutes les caractéristiques physiques de la fenêtre. Elle permet de définir si la fenêtre est redimensionnable ou modale, ainsi que ses contraintes de taille minimale et maximale.
\end{winbox}

\begin{lstlisting}
typedef struct {
    const char *title;
    const char *icon_name;
    const char *background_color;
    const char *background_image_path;
    int default_width;
    int default_height;
    bool resizable;
    bool decorated;
    bool modal;
    widget_style_config style;
} window_config;
\end{lstlisting}

\section{Logique de Personnalisation Visuelle}

\begin{winbox}{Fonction : apply\_window\_background}
Cette fonction utilise un \texttt{GtkCssProvider} pour injecter du CSS dynamiquement. Elle permet de définir une couleur de fond ou une image de fond avec la propriété \texttt{background-size: cover}, garantissant que l'image remplit toute la fenêtre.
\end{winbox}

\begin{lstlisting}
static void apply_window_background(const window_config *config, GtkWidget *window) {
    // ... Verification des parametres ...

    char *css = g_strdup_printf(
        "window#%s { background-image: url('%s'); background-size: cover; }",
        name, config->background_image_path
    );
    gtk_css_provider_load_from_string(provider, css);
    // ... Application du style au display ...
}
\end{lstlisting}

\section{Construction de la Fenêtre}

Pour calculer la surface $S$ disponible à l'écran en fonction des dimensions par défaut $w$ et $h$, on utilise simplement :
$$S = w \times h$$

\begin{winbox}{Fonction : create\_window}
C'est la fonction principale du module. Elle instancie la fenêtre liée à l'application et applique toutes les configurations (titre, icône, taille, et style CSS) avant de la présenter à l'écran.
\end{winbox}

\begin{lstlisting}
GtkWidget *create_window(GtkApplication *app, const window_config *config) {
    GtkWidget *window = gtk_application_window_new(app);
    if (config == NULL) return window;

    gtk_window_set_title(GTK_WINDOW(window), config->title);
    gtk_window_set_default_size(GTK_WINDOW(window), config->default_width, config->default_height);
    
    apply_window_background(config, window);
    apply_widget_style(window, &config->style);

    if (config->present_on_create) gtk_window_present(GTK_WINDOW(window));
    return window;
}
\end{lstlisting}
