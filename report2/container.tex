\chapter{Module de Gestion des Conteneurs}

\section{Introduction}
Dans ce module, nous gérons la disposition spatiale des éléments. Nous utilisons trois types de conteneurs principaux :
\begin{itemize}
    \item \textbf{GtkGrid} : Pour une organisation en tableau (lignes/colonnes).
    \item \textbf{GtkBox} : Pour un empilement linéaire simple.
    \item \textbf{GtkStack} : Pour gérer plusieurs pages dans une seule zone.
\end{itemize}

\section{Définitions des Configurations (Header)}

\begin{containerbox}{Structure : grid\_config}
La grille permet un alignement précis. Les options \texttt{homogeneous} permettent de forcer toutes les cellules à avoir la même taille, ce qui est utile pour les calculatrices ou les claviers numériques.
\end{containerbox}

\begin{lstlisting}
typedef struct {
    int column_spacing;
    int row_spacing;
    bool homogeneous_columns;
    bool homogeneous_rows;
    widget_style_config style;
} grid_config;

#define GRID_CONFIG_DEFAULT ((grid_config){ \
    .column_spacing = 0, \
    .row_spacing = 0, \
    .homogeneous_columns = false, \
    .homogeneous_rows = false, \
    .style = WIDGET_STYLE_CONFIG_DEFAULT, \
})
\end{lstlisting}

\section{Logique d'Implémentation (Source)}

\begin{containerbox}{Fonction : create\_box}
La Box est le conteneur le plus utilisé. Ici, nous gérons dynamiquement l'orientation (verticale ou horizontale) et l'espacement entre les widgets enfants.
\end{containerbox}

\begin{lstlisting}
GtkWidget *create_box(const box_config *config) {
    GtkWidget *box = gtk_box_new(
        config ? config->orientation : GTK_ORIENTATION_VERTICAL,
        config ? config->spacing : 0
    );
    if (config == NULL) return box;

    gtk_box_set_homogeneous(GTK_BOX(box), config->homogeneous);
    apply_widget_style(box, &config->style);
    return box;
}
\end{lstlisting}

\section{Gestion des Transitions (Stack)}
Pour le \texttt{GtkStack}, la durée de transition $t$ est définie en millisecondes. Si l'on considère $n$ pages, la navigation fluide est assurée par :
\[ \text{Transition}(p_i \rightarrow p_j) \text{ avec } \Delta t = \text{config->duration\_ms} \]

\begin{containerbox}{Fonction : create\_stack}
Le Stack est idéal pour les interfaces à onglets ou les assistants (wizards). Nous configurons le type de transition (glissement, fondu) et la durée.
\end{containerbox}

\begin{lstlisting}
GtkWidget *create_stack(const stack_config *config) {
    GtkWidget *stack = gtk_stack_new();
    if (config == NULL) return stack;

    gtk_stack_set_transition_type(GTK_STACK(stack), config->transition);
    gtk_stack_set_transition_duration(GTK_STACK(stack), config->duration_ms);
    // ... configuration de l'homogeneite ...
    apply_widget_style(stack, &config->style);
    return stack;
}
\end{lstlisting}
