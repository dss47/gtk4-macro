\chapter{Module de Commutateurs (Toggles \& Switches)}

\section{Introduction}
Le module \texttt{toggle} gère les éléments d'interface permettant à l'utilisateur de basculer entre deux états. Nous implémentons deux types de contrôles :
\begin{itemize}
    \item \textbf{GtkCheckButton} : Idéal pour les formulaires de sélection multiple.
    \item \textbf{GtkSwitch} : Utilisé pour les changements d'état immédiats (ex: activer le Wi-Fi ou un mode sombre).
\end{itemize}

\section{Définitions des Structures (Header)}

\begin{togglebox}{Structure : checkbox\_config}
Cette structure définit les propriétés d'une case à cocher, incluant son libellé et son état initial. L'état \texttt{inconsistent} permet de gérer les cas où la sélection est partielle (souvent utilisé dans les arborescences de fichiers).
\end{togglebox}

\begin{lstlisting}
typedef struct {
    const char *label;
    bool active;
    bool inconsistent;
    widget_style_config style;
    void (*on_toggled)(GtkCheckButton *button, gpointer user_data);
    gpointer user_data;
} checkbox_config;
\end{lstlisting}

\section{Logique d'Implémentation (Source)}

\begin{togglebox}{Fonction : create\_checkbox}
La fonction initialise un \texttt{GtkCheckButton}. Si une fonction de rappel (\textit{callback}) est fournie, elle connecte le signal "toggled", permettant de réagir instantanément au clic de l'utilisateur.
\end{togglebox}

\begin{lstlisting}
GtkWidget *create_checkbox(const checkbox_config *config) {
    GtkWidget *checkbox = gtk_check_button_new_with_label(config && config->label ? config->label : "");
    if (config == NULL) return checkbox;

    gtk_check_button_set_active(GTK_CHECK_BUTTON(checkbox), config->active);
    apply_widget_style(checkbox, &config->style);

    if (config->on_toggled != NULL) {
        g_signal_connect(checkbox, "toggled", G_CALLBACK(config->on_toggled), config->user_data);
    }
    return checkbox;
}
\end{lstlisting}

\section{Gestion des Interrupteurs en Ligne}



\begin{togglebox}{Fonction : create\_switch\_row}
Contrairement à la case à cocher simple, cette fonction crée une ligne complète (\texttt{GtkBox}) contenant un label à gauche et un interrupteur à droite. Cela garantit un alignement parfait dans les menus de paramètres.
\end{togglebox}

\begin{lstlisting}
GtkWidget *create_switch_row(const switch_config *config, GtkWidget **out_switch) {
    GtkWidget *row = gtk_box_new(GTK_ORIENTATION_HORIZONTAL, 10);
    GtkWidget *sw = gtk_switch_new();
    
    // Organisation spatiale
    gtk_widget_set_hexpand(label, TRUE);
    gtk_widget_set_halign(label, GTK_ALIGN_START);
    
    if (config->on_active_changed != NULL) {
        g_signal_connect(sw, "notify::active", G_CALLBACK(config->on_active_changed), config->user_data);
    }
    
    if (out_switch != NULL) *out_switch = sw;
    return row;
}
\end{lstlisting}
