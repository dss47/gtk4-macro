\section{Module \texttt{menu}}

Le module \texttt{menu} construit des menus hiérarchiques et enregistre les actions GTK associées.

\subsection{Macros de Configuration}

\begin{docbox}{Macros \texttt{MENU\_ITEM\_CONFIG\_DEFAULT}, \texttt{MENU\_SECTION\_CONFIG\_DEFAULT}, \texttt{MENUBAR\_CONFIG\_DEFAULT}}
\begin{lstlisting}[language=C]
#define MENU_ITEM_CONFIG_DEFAULT ((menu_item_config){ \
    .label = NULL, \
    .kind = MENU_ITEM_ACTION, \
    .children = NULL, \
    .child_count = 0, \
    .enabled = true, \
    .on_activate = NULL, \
    .user_data = NULL, \
})

#define MENU_SECTION_CONFIG_DEFAULT ((menu_section_config){ \
    .label = NULL, \
    .items = NULL, \
    .item_count = 0, \
})

#define MENUBAR_CONFIG_DEFAULT ((menubar_config){ \
    .sections = NULL, \
    .section_count = 0, \
    .action_map = NULL, \
    .layout = MENU_LAYOUT_HORIZONTAL, \
    .show_arrow_indicators = true, \
    .style = WIDGET_STYLE_CONFIG_DEFAULT, \
})
\end{lstlisting}
\end{docbox}

Ces macros posent une base d'arbre de menu vide mais valide. Les compteurs à \texttt{0} et pointeurs \texttt{NULL} empêchent les parcours hors bornes et les déréférencements invalides. C'est particulièrement important dans ce module qui manipule la récursivité.

Ces valeurs par défaut sont aussi essentielles pour stabiliser l'algorithme récursif de construction des sous-menus. Une initialisation partielle peut facilement provoquer des accès mémoire invalides; les macros réduisent ce risque.

\subsection{Fonctions API}

\begin{lstlisting}[language=C]
GtkWidget *create_menubar(const menubar_config *config) {
    GMenuModel *model = build_menu_model(config);
    GtkWidget *bar = gtk_popover_menu_bar_new_from_model(model);
    if (config != NULL) {
        apply_widget_style(bar, &config->style);
    }
    g_object_unref(model);
    return bar;
}

GtkWidget *create_menu_panel(const menubar_config *config) {
    GMenuModel *model = build_menu_model(config);
    GtkWidget *panel = gtk_popover_menu_new_from_model(model);
    g_object_unref(model);
    return panel;
}
\end{lstlisting}

Le module sépare la description logique du menu (structures) de la génération GTK (\texttt{GMenuModel} + widgets).

La fonction \texttt{create\_menubar} sert de point d'entrée unique: elle construit le modèle puis instancie le widget final. Cette séparation améliore la lisibilité et facilite l'ajout de nouvelles actions.
