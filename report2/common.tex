\chapter{Module de Style Commun (Common)}

\section{Introduction}
Ce module définit les bases de l'apparence visuelle de l'application. Au lieu de configurer manuellement chaque propriété GTK pour chaque widget, nous utilisons une structure de configuration centralisée. Cela permet de respecter les principes de programmation propre (DRY - Don't Repeat Yourself).

\section{Structure de Données}

\begin{docbox}{Structure : widget\_style\_config}
Cette structure regroupe les propriétés de mise en page les plus courantes : les marges, l'alignement, l'expansion (hexpand/vexpand) et les classes CSS. Elle permet de définir le "look and feel" d'un composant de manière déclarative.
\end{docbox}

\begin{lstlisting}
typedef struct {
    const char *css_class;
    const char **css_classes;
    int width_request;
    int height_request;
    int margin_top;
    int margin_bottom;
    int margin_start;
    int margin_end;
    GtkAlign halign;
    GtkAlign valign;
    bool set_halign;
    // ... expansion et visibilite ...
    const char *tooltip;
} widget_style_config;
\end{lstlisting}

\section{Macros de Sécurité}

\begin{docbox}{Macro : WIDGET\_STYLE\_CONFIG\_DEFAULT}
L'objectif est d'obtenir une configuration neutre (zéro-initialisée). En utilisant \texttt{\{0\}}, tous les pointeurs deviennent \texttt{NULL} et les entiers \texttt{0}, ce qui permet de ne modifier ensuite que les champs nécessaires.
\end{docbox}

\begin{lstlisting}
#define WIDGET_STYLE_CONFIG_DEFAULT ((widget_style_config){0})
\end{lstlisting}

\section{Logique d'Application du Style}

\begin{docbox}{Fonction : apply\_widget\_style}
Cette fonction fait le pont entre notre structure personnalisée et l'API native de GTK4. Elle vérifie chaque champ de la structure et appelle les fonctions \texttt{gtk\_widget\_set\_*} correspondantes.
\end{docbox}

\begin{lstlisting}
void apply_widget_style(GtkWidget *widget, const widget_style_config *style) {
    if (widget == NULL || style == NULL) return;

    // Gestion des classes CSS
    if (style->css_class && style->css_class[0] != '\0') {
        gtk_widget_add_css_class(widget, style->css_class);
    }

    // Gestion des dimensions et marges
    if (style->width_request != 0 || style->height_request != 0) {
        gtk_widget_set_size_request(widget, style->width_request, style->height_request);
    }

    gtk_widget_set_margin_top(widget, style->margin_top);
    // ... configuration des alignements et tooltips ...
}
\end{lstlisting}
